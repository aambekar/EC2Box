About EC2Box

A web-based ssh console to execute commands and manage multiple EC2 instances simultaneously running on Amazon Web Services (AWS). EC2Box allows you to share terminal commands and upload files to all your EC2 instances. Once the sessions have been opened you can select a single EC2 instance or any combination to run your commands. Also, additional instance administrators can be added and their terminal sessions and history can be audited.

Composite SSH terminals. Total control.

Execute commands on multiple instances simultaneously. Upload files to selected instances. Once the sessions have been opened you can select a single instance or any combination to run your commands.

Control user access. Secure your instances.

Create users with limited privileges and audit their administrative activities.

Import EC2 Keys

Once you have registered your EC2 SSH key, all instances associated with the key will show on the 'Composite SSH Terminal' screen. Instances will only show if the key assocated with the instance has been imported.

Upload files with ease to selected EC2 instances. Files can be uploaded to all instances or any combination of instances.
